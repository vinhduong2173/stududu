Ô nhập liệu chuẩn (bo tròn 12px, focus ring primary). Prop `error` hiển thị helper text đỏ + viền đỏ. Nhận mọi props của `<input>` (type, placeholder, value…).

```tsx
<Input type="email" placeholder="Email" />
<Input type="password" placeholder="Mật khẩu" error="Tối thiểu 8 ký tự, gồm chữ và số" />
```
